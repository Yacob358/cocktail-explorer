# Cocktail Explorer

This is a web application where users can search cocktails and save their favorites.

## Features

- Search cocktails using CocktailDB API
- Show cocktail name and image
- Save drinks to favorites
- View saved drinks on a separate page

## Technologies

- HTML
- CSS
- JavaScript
- Bootstrap
- CocktailDB API
- localStorage

## How to use

1. Open index.html
2. Type cocktail name
3. Click Search
4. Add drink to favorites
5. Open Favorites page to see saved drinks
